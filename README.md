# Web Beta - Dự án Website Bán Hàng Fullstack

## Giới thiệu
Web Beta là dự án website bán hàng fullstack được xây dựng nhằm phục vụ học tập, làm đồ án và phát triển portfolio cá nhân.  
Dự án tập trung vào việc xây dựng một hệ thống bán hàng hoàn chỉnh gồm giao diện người dùng, quản trị admin, xử lý đơn hàng, cơ sở dữ liệu và tích hợp thanh toán.

## Chức năng chính
- Xem danh sách sản phẩm theo danh mục
- Xem chi tiết sản phẩm
- Tìm kiếm sản phẩm
- Thêm sản phẩm vào giỏ hàng
- Đặt hàng và thanh toán
- Đăng ký / đăng nhập tài khoản
- Quản lý thông tin cá nhân và đơn hàng
- Trang quản trị admin: sản phẩm, đơn hàng, dashboard
- Tích hợp thanh toán MoMo

## Công nghệ sử dụng
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js, Java Spring Boot
- **Database:** PostgreSQL
- **Công cụ khác:** Git, GitHub, Maven, MoMo Payment

## Cấu trúc dự án
- `client/`: giao diện người dùng
- `server/`: backend Node.js / Express
- `api/`: backend Java Spring Boot
- `database/`: file SQL và cấu trúc dữ liệu

## Điểm nổi bật
- Xây dựng hệ thống bán hàng fullstack hoàn chỉnh
- Có cả giao diện người dùng và quản trị viên
- Tích hợp thanh toán MoMo
- Làm việc với PostgreSQL và xử lý dữ liệu thực tế
- Sử dụng song song Node.js và Java Spring Boot trong cùng hệ thống

## Hướng phát triển
- Hoàn thiện responsive trên mọi thiết bị
- Tối ưu giao diện và trải nghiệm người dùng
- Bổ sung kiểm thử và bảo mật hệ thống
- Triển khai demo online trong tương lai
